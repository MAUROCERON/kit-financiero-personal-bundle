# Free CSV Expense Analyzer + Personal Finance Spreadsheet Bundle

![Finance Kit logo](https://mauroceron.github.io/kit-financiero-personal-bundle/assets/logo-kit-financiero.png)

A lightweight personal finance toolkit for people who want to understand expenses without setting up heavy accounting software.

## Try The Free Tool

Use the free browser-based analyzer:

https://mauroceron.github.io/kit-financiero-personal-bundle/csv-expense-analyzer.html

Paste or upload a CSV and get:

- Income, expenses and available cash.
- Savings rate.
- Top spending categories.
- Possible subscription signals.
- A quick read that stays local in your browser.

No bank login, password or card number is required.
